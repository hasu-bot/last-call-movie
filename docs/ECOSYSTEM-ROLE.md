# 映画『ラストコール』— エコシステム内での役割と展開計画

> 天草を舞台にした青春・音楽映画『ラストコール』の、Creative YOLO エコシステムにおける位置づけ。
> 上位計画：yolo-members `docs/creative-yolo/00-MASTER-PLAN.md` §2。

---

## 1. 役割：旗艦作品（フラッグシップ）

ラストコールは単なる1作品ではなく、**「YOLOは本当に作品を作れる」という証明**。

- **依頼者への証明**：相談サイトに「映像を頼みたい」人が来たとき、実績として見せられる最上位の名刺
- **メンバーへの証明**：「コミュニティから長編（級）の作品が生まれる」という前例。次の企画の心理的ハードルを下げる
- **市民への入口**：映画を観た人が YOLO を知る最初の接点になり得る

## 2. サイト改善タスク（UX監査 P0 含む）

| 優先 | 内容 |
|---|---|
| **P0** | `og:image` が相対パス（`assets/photos/key-visual.jpg`）のため **SNSシェアでキービジュアルが表示されない**。絶対URLへ差し替え＋ `og:url` 追加。映画サイトの流入はシェア経由が主力のため最優先 |
| P1 | Creative YOLO 共通フッター追加（現状、エコシステムへの外部リンクなし）。特に「製作：Creative YOLO」→ポータル、「キャスト・スタッフ」→MEMBERS 名鑑リンク |
| P2 | 上映情報の更新運用：上映・配信の予定が動くたびにサイト冒頭へ。終映後は「アーカイブ＋次の観られる機会」へ書き換え |

## 3. 展開計画

| 時期 | 展開 |
|---|---|
| 2026 | 上映機会の確保（電気館・天草での凱旋上映・自主上映会）。上映後は必ず Discord/MAGAZINE 告知（循環接続） |
| 2026-27 | MAGAZINE で「制作の裏側」連載——**制作プロセス自体をコンテンツ化**し、「自分も作りたい」人（ペルソナA）の入口にする |
| 2027 | 映画祭との連動企画（特別上映・監督トーク）。「映画祭から次のラストコールが生まれる」物語を作る |
| 2028〜 | 次回作の企画。ラストコールで確立した座組（キャスト・スタッフ・天草ロケ）を YOLO OS に記録し、再現可能にする |

## 4. 記録すべきもの（次回作のための資産化）

作品そのものと同じくらい、**「どう作ったか」が資産**。散逸する前に記録する：

- 座組と役割分担（誰が何を担ったか）
- 予算の実績（費目レベルでよい）
- ロケ地リスト・許可取りの連絡先（天草）
- 音楽・肖像などの権利処理の実例
- うまくいかなかったこと（次回作の SOP の種）

→ 記録先はこの `docs/` に `PRODUCTION-NOTES.md` として追加（非公開情報は運営内共有に留める）。
